# 📋 Project PRD: 팀허브(TeamHub)

## 1. 핵심 기능 (Core Features) 

* **기능 A (실시간 소통 및 공지):** 디스코드처럼 채팅과 공지 기능 제공 / 파일 및 링크를 공유하여 모든 프로젝트 자료를 한곳에서 관리[cite: 1]
* **기능 B (역할·일정 관리):** 프로젝트 생성 및 팀원 초대 / 역할 분담, 일정, 마감 기한, 과제 제출 현황을 한 화면에서 관리[cite: 1]
* **기능 C (몰입 타이머 및 인증):** 열품타 방식의 몰입 타이머 제공 / 실시간 작업 인증과 진행률 공유를 통해 프로젝트 진행 과정을 투명하게 확인[cite: 1]

---

## 2. 사용자 흐름 (User Flow)

1. 앱 접속 후 팀 프로젝트 방 입장[cite: 1]
2. 새로 올라온 공지 및 마감 기한을 확인한 뒤 `[확인 완료]` 버튼 클릭[cite: 1]
3. 내장된 실시간 채팅으로 팀원들과 소통하며 과제 시작, 몰입 타이머를 켜고 화면 잠금 상태로 집중 작업 수행[cite: 1]
4. 작업 완료 후 파일 저장소에 과제물 업로드 및 본인의 진척도 갱신[cite: 1]
5. 메인 대시보드에서 팀원들의 공지 확인 여부, 누적 시간, 진척도 확인 후 과제 최종 완성[cite: 1]

## 3. 화면 구성 및 화면별 기능

| 화면 이름 | 화면의 역할 | 주요 기능 (Lovable 명령용) |
| :--- | :--- | :--- |
| **1. 메인 대시보드 화면** | 팀 프로젝트의 현황을 한눈에 모니터링하는 화면[cite: 1] | * 왼쪽 영역: [팀원별 실시간 진척도 그래픽 바], [누적 집중 시간], [공지 확인 여부(체크 표시)]가 시각적으로 나타남.[cite: 1]<br>* 오른쪽 영역: 팀장이 올린 [최신 공지사항 카드]와 [다가오는 마감 일정 요약 리스트]가 나열됨.[cite: 1]<br>* 공지 카드를 클릭하면 '공지 상세 팝업창'이 열려야 함.[cite: 1] |
| **2. 공지 확인 팝업 화면** | 공지 세부 내용을 읽고 수락하는 화면[cite: 1] | * 팀장이 작성한 역할 분담, 마감일 등의 세부 텍스트가 표시됨.[cite: 1]<br>* 하단에 [공지 확인 완료] 버튼이 있음: 버튼을 누르면 해당 팀원의 대시보드 상태가 '확인 완료(체크)'로 실시간 변경되며 메인 화면으로 돌아감.[cite: 1] |
| **3. 캘린더 (일정) 화면** | 프로젝트의 전체 일정과 마감일을 한눈에 확인하는 화면[cite: 1] | * 월간(Month) 달력 뷰가 화면 중앙에 큼직하게 렌더링됨.[cite: 1]<br>* [일정 등록] 버튼: 클릭하면 일정 이름, 담당 팀원, 마감 기한을 입력해 달력에 색상 카드로 추가할 수 있음.[cite: 1]<br>* 달력의 특정 날짜나 일정을 클릭하면 누가 언제까지 해야 하는 역할인지 상세 팝업이 뜸.[cite: 1] |
| **4. 워크스페이스 화면 (채팅&자료실)** | 한 공간에서 대화하고 자료를 보관하는 화면[cite: 1] | * 왼쪽 영역: 실시간 메시지 입력과 전송이 가능한 [채팅창]이 작동함.[cite: 1]<br>* 오른쪽 영역: 구글 드라이브 스타일의 [자료 보관함]이 있으며, [파일 업로드] 버튼을 누르면 로컬 파일이 목록에 등록되고 누구나 클릭해 다운로드할 수 있음.[cite: 1] |
| **5. 몰입 타이머 화면** | 열품타처럼 딴짓을 방지하며 집중 시간을 측정하는 화면[cite: 1] | * 화면 중앙에 큰 [집중 시작] 버튼이 있음.[cite: 1]<br>* [집중 시작] 버튼 클릭 시: 타이머가 흐르며 주변 화면이 어두워지는 잠금 모드가 활성화됨.[cite: 1]<br>* [집중 종료] 버튼 클릭 시: 타이머가 멈추고 [진척도 입력 창]과 [인증샷 업로드] 창이 나타남. 입력 후 [저장]을 누르면 대시보드에 반영됨.[cite: 1] |

## 4. 구현 범위 및 제외 사항 (Scope)

### ✅ 구현 범위 (만들 기능)
* **회원가입 및 로그인:** 단일 테스트 계정 또는 간이 로그인으로 접속 기능[cite: 1]
* **프로젝트 생성 및 팀원 초대:** 수행평가 조별 방을 만들고 팀원을 임의로 추가하는 화면[cite: 1]
* **공지 및 역할 분담:** 노션 스타일의 공지 작성 및 공지 읽음(`[확인 완료]`) 버튼 기능[cite: 1]
* **캘린더 및 일정 관리:** 월간 달력 뷰에서 일정을 등록하고 담당자와 마감일을 표시하는 기능[cite: 1]
* **실시간 채팅:** 워크스페이스 내에서 팀원 간 텍스트 메시지를 주고받는 소통 창 기능[cite: 1]
* **자료(파일·링크) 공유:** 구글 드라이브 스타일의 가상 파일 업로드 및 링크 저장 보관함 기능[cite: 1]
* **과제 제출 및 진행률 확인:** 개인별 수행 진척도를 메인 대시보드에 막대 그래프로 시각화하는 기능[cite: 1]
* **몰입 타이머 및 작업 인증 기능:** 열품타 스타일의 타이머 구동, 화면 잠금 흉내, 종료 시 인증샷 및 진척도 입력 연동 기능[cite: 1]

### ❌ 제외 사항 (만들지 않을 기능)
* 음성·화상 회의 기능 / 실시간 화면 공유 기능[cite: 1]
* AI 자동 과제 생성 기능[cite: 1]
* 외부 메신저(카카오톡, 디스코드) 연동[cite: 1]
* 프로젝트 통계 및 분석 기능 / 데이터 영구 저장 (실시간 클라우드 서버 연동)[cite: 1]
* 푸시 알림 기능 (추후 개발 예정)[cite: 1]
* 실제 디바이스 전체 제어 잠금 (웹브라우저 내 잠금 화면 UI로만 대체)[cite: 1]

> 
> * **팀원:** 1208 여서정, 1317 함서희
> * **작성일자:** 2026-07-14

## 📌 프로젝트 기획 개요

### 💡 해결하려는 문제 (불편사항)
팀 프로젝트를 진행할 때 공지, 소통, 자료 공유, 일정 관리 등을 서로 다른 플랫폼에서 사용하는 경우가 많아 정보가 분산되는 문제가 발생합니다. 이로 인해 팀원들이 자신의 역할이나 마감 기한을 정확하게 확인하지 못하거나 공지를 놓치고 연락을 확인하지 않아 과제를 제때 수행하지 않는 상황이 자주 발생합니다.[cite: 2]

### 🎯 서비스 개요
본 서비스는 팀 프로젝트와 수행평가 진행 시 발생하는 정보 분산, 연락 두절, 무임승차 문제를 해결하기 위한 학생 전용 올인원 협업 관리 웹애플리케이션입니다.[cite: 2] 기존의 여러 플랫폼(노션, 디스코드, 구글 드라이브, 열품타)에 파편화되어 있던 핵심 기능들을 하나로 통합하여 팀원 간의 소통 효율을 극대화합니다.[cite: 2]

### 👥 핵심 사용자
* 팀워크가 필요한 단체 수행평가와 팀 프로젝트를 진행하는 중·고등학생 및 대학생 등 협업이 필요한 모든 학생[cite: 1, 2]

## 🔍 기존 서비스 조사 및 차별점

### 📊 기존 서비스의 장단점
* **노션 (Notion)**
  * **장점:** 공지, 일정, 역할을 깔끔하게 정리 가능함.[cite: 2]
  * **단점:** 기능이 너무 복잡하고, 팀원이 공지를 진짜 읽었는지 확인 불가함.[cite: 2]
* **구글 드라이브**
  * **장점:** 과제 자료 보관과 실시간 공동 편집이 편리함.[cite: 2]
  * **단점:** 파일이 쌓이면 찾기 어렵고, 실제 작업 진척도를 알 수 없음.[cite: 2]
* **열품타**
  * **장점:** 폰 잠금과 타이머로 딴짓을 막고 집중을 도움.[cite: 2]
  * **단점:** 시간만 잴 뿐, 그 시간에 진짜 과제를 했는지 결과물 증명이 안 됨.[cite: 2]
* **디스코드**
  * **장점:** 실시간 채팅과 음성 통화로 빠른 소통과 회의가 가능함.[cite: 2]
  * **단점:** 대화가 쌓이면 공지와 파일이 묻히고, 알림을 끄면 연락 두절됨.[cite: 2]

### ✨ 우리 서비스의 차별점
노션처럼 공동 프로젝트를 관리할 수 있는 기능과 디스코드처럼 실시간으로 소통할 수 있는 기능을 하나의 플랫폼에서 제공합니다.[cite: 2] 또한 과제 제출 일정, 팀원별 역할, 마감 기한 등을 한눈에 확인할 수 있어 프로젝트 진행 상황을 체계적으로 관리할 수 있습니다.[cite: 2] 여기에 열품타와 같은 몰입 타이머와 실시간 작업 진척도 인증 기능을 제공하여 단순히 “과제를 하고 있다"고 말하는 것이 아니라 실제 작업 과정을 투명하게 공유할 수 있어 무임승차를 효율적으로 방지합니다.[cite: 2]
